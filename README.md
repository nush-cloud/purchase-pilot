# Purchase Pilot

Purchase Pilot is an AI-powered shopping assistant built with Next.js, TypeScript, Bootstrap, React-Bootstrap, and Anthropic API.

## Goal
Help users discover the right products faster through conversational recommendations, comparison, and saved items.

## Tech Stack
- Next.js
- React
- TypeScript
- Bootstrap
- React-Bootstrap
- Anthropic API
- GitHub
- Vercel
- Figma

## Current Status
Project setup complete. Bootstrap integration in progress.

## Planned Features
- Conversational shopping input
- Follow-up questions
- Product recommendation cards
- Compare products
- Save products
- Loading, error, and empty states
