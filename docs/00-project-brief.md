# Project Brief — Purchase Pilot

## Problem
Online shopping is often overwhelming because users must compare many products, tabs, reviews, and specifications before making a decision.

## Goal
Build an AI shopping assistant that asks helpful follow-up questions and gives guided product recommendations.

## Target Users
Students and young professionals who want quick, trustworthy shopping guidance.

## MVP Features
- User enters a shopping need
- AI asks follow-up questions
- App shows product recommendations
- User can save products
- User can compare products

## Non-Goals
- Checkout
- Payments
- Authentication
- Production ecommerce backend
- Large product catalog

## Success Criteria
- App runs locally
- Bootstrap UI is set up correctly
- User can complete a basic recommendation flow
- Project is documented for GitHub and portfolio case study
