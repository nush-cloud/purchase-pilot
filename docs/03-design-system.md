# Design System Notes

## UI Direction
Clean, modern, trustworthy, easy to scan.

## Foundations
- Bootstrap grid for layout
- Cards for product recommendations
- Forms for user input
- Buttons for actions
- Tables for product comparison

## Visual Goals
- Clear hierarchy
- Good spacing
- Minimal clutter
- Mobile-friendly design

## Chat Patterns
- assistant messages use light cards
- user messages use primary-colored cards
- input area stays fixed at the bottom of the chat card
