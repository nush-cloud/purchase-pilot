# User Flow

## Main Flow
1. User opens Purchase Pilot
2. User enters shopping need
3. AI asks follow-up questions
4. User answers
5. App shows product recommendations
6. User saves or compares products

## Saved Flow
1. User opens saved page
2. Reviews saved products
3. Selects products to compare

## Compare Flow
1. User opens compare page
2. Reviews side-by-side product information
3. Chooses the best option
